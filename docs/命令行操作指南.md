# 命令行操作指南

## 🖥️ 如何打开终端（Terminal）

### macOS 方法：

1. **方法 1：使用 Spotlight 搜索**
   - 按 `Command + Space`（空格键）
   - 输入 `Terminal` 或 `终端`
   - 按 Enter 打开

2. **方法 2：从应用程序打开**
   - 打开"应用程序"文件夹
   - 找到"实用工具"文件夹
   - 双击"终端"应用

3. **方法 3：使用 Finder**
   - 打开 Finder
   - 按 `Command + Shift + U` 打开"实用工具"
   - 双击"终端"

## 📝 基本命令操作

### 1. 进入项目目录

```bash
cd "/Users/balala/个人资料/Blog/MyBlog"
```

**说明**：
- `cd` = change directory（切换目录）
- 路径要用引号包起来（因为路径中有中文和空格）

### 2. 查看当前目录

```bash
pwd
```

**说明**：显示当前所在的目录路径

### 3. 列出文件

```bash
ls
```

**说明**：显示当前目录下的文件和文件夹

## 🚀 修复 Git Remote 的完整步骤

### 步骤 1: 打开终端

按 `Command + Space`，输入 `Terminal`，按 Enter

### 步骤 2: 进入项目目录

在终端中输入（可以复制粘贴）：

```bash
cd "/Users/balala/个人资料/Blog/MyBlog"
```

按 Enter 执行

### 步骤 3: 查看当前 remote 配置

```bash
git remote -v
```

按 Enter 执行

### 步骤 4: 修复 remote URL

**方法 A：使用修复脚本（推荐）**

```bash
./fix-git-remote.sh
```

按 Enter 执行

**方法 B：手动修复**

```bash
git remote set-url origin https://github.com/WiseInvest513/WiseInvest.git
```

按 Enter 执行

### 步骤 5: 验证修复

```bash
git remote -v
```

按 Enter 执行

应该看到：
```
origin	https://github.com/WiseInvest513/WiseInvest.git (fetch)
origin	https://github.com/WiseInvest513/WiseInvest.git (push)
```

### 步骤 6: 先试试直接推送（不用代理）

**重要**：先试试不用代理，如果网络正常就不需要代理！

```bash
git push origin main
```

**结果判断**：
- ✅ **如果成功**：恭喜！不需要代理，以后直接推送即可
- ❌ **如果超时/失败**：继续步骤 7（配置代理）

### 步骤 7: 如果超时，配置代理

**只有在上一步失败时才需要这一步！**

```bash
# 设置代理
export http_proxy=http://127.0.0.1:7897
export https_proxy=http://127.0.0.1:7897

# 再次推送
git push origin main
```

## 💡 命令行小技巧

### 1. 复制粘贴
- **复制**：选中文本，按 `Command + C`
- **粘贴**：在终端中按 `Command + V`

### 2. 自动补全
- 输入文件或目录名的前几个字母
- 按 `Tab` 键自动补全

### 3. 查看历史命令
- 按 `↑`（上箭头）查看之前输入的命令
- 按 `↓`（下箭头）向下查看

### 4. 清屏
```bash
clear
```
或按 `Command + K`

### 5. 中断命令
- 如果命令卡住了，按 `Control + C` 中断

## 📋 完整操作示例

### 示例 1: 最简单的方式（先试试不用代理）

```bash
# 1. 打开终端后，进入项目目录
cd "/Users/balala/个人资料/Blog/MyBlog"
```

## ⚠️ 常见问题

### Q: 提示 "command not found"
**A**: 可能是：
- 命令拼写错误
- 没有进入正确的目录
- 文件不存在

### Q: 提示 "Permission denied"
**A**: 
- 检查文件权限：`ls -l fix-git-remote.sh`
- 如果没有执行权限：`chmod +x fix-git-remote.sh`

### Q: 路径中有空格怎么办？
**A**: 用引号包起来：
```bash
cd "/Users/balala/个人资料/Blog/MyBlog"
```

### Q: 如何退出终端？
**A**: 
- 关闭窗口：`Command + Q`
- 或者输入 `exit` 然后按 Enter
